Command Line for beginners

Today I learned command line, which is quite interesting. I have learned lot of basic and advance commands. What I was aware but did not use since long time. Learning commands was again a good experience for me, it is always good to learn and enhance the skill set.

Following are the basic commands which is useful for a beginners-

Command
Use
Pwd (print working directory)
Gives current path                                   
hostname
Gives computer name
mkdir(Make A Directory)
Create new directory
cd(Change Directory)
Changing into directories using path
ls(list Directory)
lists out the contents of the directory you are currently in
rmdir(Remove Directory)
It is used to remove the directory
pushd
The pushd command takes your current directory and "pushes" it onto a list for later
popd
The popd command takes the last directory you pushed and "pops" it off, taking you back there.
touch
Makes Empty Files
cp
Copy A File
mv
Move file
less
View file
cat
This command just spews the whole file to the screen with no paging or stopping.
This command is also useful for concatenating two and more files
rm
Remove file
               
I like wild card searching since it allows us to filer the file list based on specified input.
For example-
·    While dealing with CD (Change Directory) commands. Type "*" after writing few characters and   press enter.  You will get redirected to the unique Folder that starts with those input character
 C:\>cd prog*
 C:\Program Files
·     While dealing with ls (List Directory) commands. Type "*" before writing “.txt” and press enter.  You will get the list of content of in the current directory, which ends with “.txt”.
          ls *.txt
·     rm *.txt to remove all of the .txt files in the current directory.               
·     cat *.txt > new.txt concatenate all the txt files data, present in the current directory to a new txt file “new”.
